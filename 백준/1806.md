## 부분합

- 투포인터

while 문에서는 left만 체크한다.

먼저 정답 체크하고

right가 끝에 갔는지 확인하고 left만 움직인다.

아니라면 조건에 맞게 left와 right를 움직인다.

## 코드

```cpp
#include <algorithm>
#include <cmath>
#include <cstring>
#include <iomanip>
#include <iostream>
#include <limits>
#include <map>
#include <numeric>
#include <queue>
#include <set>
#include <stack>
#include <string>
#include <vector>
#include <tuple>

using namespace std;

int n, s;

int main()
{
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    cin >> n >> s;
    vector<int> v(n);

    for (int i = 0; i < n; ++i) {
        cin >> v[i];
    }
    int ans = numeric_limits<int>::max();
    int left = 0;
    int right = 0;
    int sum = v[0];

    while (left != n) {
        if (sum >= s) ans = min(right - left + 1, ans);

        if (right == n - 1) {
            sum -= v[left];
            left++;
            continue;
        }

        if (sum > s) {
            sum -= v[left];
            left++;
        }
        else {
            right++;
            if (right < n) sum += v[right];
        }
    }
    if (ans == numeric_limits<int>::max()) ans = 0;
    cout << ans;

}
```
